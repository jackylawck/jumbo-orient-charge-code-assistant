# 東淦工程有限公司 (Jumbo Orient)
### 🏗️ 內部地盤扣帳方與合規查詢智能助理

本工具專為東淦前線管理人員、地盤總管（PM/管工）及合約管理團隊設計，旨在透過本地智能語意技術，協助團隊快速翻查公司內部《如何選擇扣賬方》指引，精確釐定突發地盤情境的扣帳代號（如 JOA / JOI / JOM），提升財務審計合規效率。

## 💡 核心功能與設計亮點

* **高頻情境直擊：** 內置地盤高頻突發事件（例如：非吸煙區食煙遭主判罰款、牛房門被撞爛等）的快速判定網閘，一秒鎖定責任方。
* **本地語意翻查：** 採用純本地比對引擎（FAISS 向量庫），只要在左側上傳指引 PDF，即可用日常廣東話自由輸入提問，自動撈出相關度最高的原始條文。
* **100% 隱私安全保障：** 本系統**完全不設外部 AI 接口**，所有上傳的文件只會加載於當前會話的暫存記憶體內，**網頁關閉或重新整理後即刻全量徹底銷毀**，絕對不會儲存到互聯網上或本代碼庫中，確保公司資料零外泄。

## 🛠️ 使用說明書（前線三步到位）

1. **上傳知識庫：** 打開網頁連結後，在左側邊欄上傳東淦官方的《如何選擇扣賬方.pdf》或最新題庫文件。
2. **輸入情境：** 在下方的對話框中，用廣東話輸入您遇到的地盤真實情況（例如：「施工組落單買錯物料要做執修點計？」）。
3. **查閱結果：** 系統會為您精準導讀對應的指引出處，並附帶系統 Remark 填寫範例。
