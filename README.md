## 💡 核心功能與設計亮點

* **高頻情境直擊：** 內置地盤高頻突發事件的快速判定網閘，一秒鎖定責任方。
* **雙驅動智能翻查：** 同事可以直接將公司現成的 **Excel 題庫 (.xlsx)** 或 **指引 PDF** 拖入系統。支持日常廣東話或地盤簡稱輸入，系統會自動在後台進行模糊語意對齊，精準撈出對應的原始紀錄。
* **100% 數據隱私安全：** 本系統**完全不連接任何外部互聯網 AI 接口**，您上傳的 Excel 或 PDF 文件只會暫存在當前網頁的臨時記憶體內。**網頁一關閉或重新整理，所有數據就會立即全量徹底銷毀**，絕對不會儲存到網絡上，確保公司資料零外泄。
