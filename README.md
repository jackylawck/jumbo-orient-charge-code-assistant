# 東淦工程有限公司 (Jumbo Orient)
### 🏗️ 內部地盤扣帳方與合約指引智能查詢助理

本工具專為東淦前線管理人員、地盤總管（PM/管工）及合約管理團隊設計。透過簡單的本地文字對齊技術，協助團隊快速翻查公司內部《如何選擇扣賬方》指引，精確釐定突發地盤情境的扣帳代號（如 JOA / JOI / JOM），提升日常工作效率。

👉 **[點擊此處直接打開：東淦智能助理網頁版連結](https://jo-charge-code-assistant.streamlit.app/)**

---

## 💡 核心功能與設計亮點

* **高頻情境直擊：** 內置地盤高頻突發事件（例如：牛房門被撞爛等）的快速判定網閘，一秒鎖定責任方。
* **本地描述翻查：** 只要在左側上傳指引 PDF，即可用日常廣東話輸入地盤情況，系統會自動在文件內撈出相關度最高的原始條文。
* **100% 數據隱私安全：** 本系統**完全不連接任何外部互聯網 AI 接口**，您上傳的文件只會暫存在當前網頁的臨時記憶體內。**網頁一關閉或重新整理，文件就會立即全量徹底銷毀**，絕對不會儲存到網絡上，確保公司資料零外泄。

---

## 🛠️ 使用說明書（前線三步到位）

1. **上傳指引：** 打開網頁連結後，在左側邊欄上傳東淦官方的《如何選擇扣賬方.pdf》文件。
2. **輸入情境：** 在下方的對話框中，用廣東話輸入您遇到的地盤真實情況（例如：「施工組落單買錯物料要做執修點計？」）。
3. **查閱結果：** 系統會為您精準導讀對應的指引出處，並附帶系統 Remark 填寫範例。
